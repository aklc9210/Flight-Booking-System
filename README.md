# Phần mềm đặt vé máy bay

## Note
Nếu dùng Windows, nhớ chỉnh path lại cho đúng

## Setup
### Yêu cầu
- Python 3.11.9

### Cài đặt
```bash
$ pip install -r client_app/requirements.txt
$ pip install -r webservice/requirements.txt
```

## Chạy
### web service
```bash
$ cd webservice
$ python run.py
```

### client app
```bash
$ cd client_app
$ python run.py
```
